# 單檔分享頁：複製連結按鈕 — 實作計劃

## 目標
1. 新建一個獨立單檔網頁 `share.html`（本計劃新建，不依賴任何既有程式碼或資源）：頁面顯示目前網址與一顆「複製連結」按鈕
2. 點擊按鈕把網址寫入剪貼簿；成功顯示「已複製」提示 2 秒；失敗（權限拒絕或 API 不支援）顯示「複製失敗，請手動複製」

## 實作步驟

### Step 1｜頁面骨架（新建 share.html）
- 單一 HTML 檔內含全部 markup／CSS／JS，無任何外部資源
- 頁面載入時把 `window.location.href` 顯示在唯讀輸入框

### Step 2｜複製邏輯
- 按鈕點擊呼叫 `navigator.clipboard.writeText(...)`：resolve → 顯示「已複製」、2000ms 後隱藏；reject 或 API 不存在 → 顯示「複製失敗，請手動複製」
- 連續點擊：重設計時器，提示不疊加

### Step 3｜驗收
- 開啟 share.html、點按鈕 → 貼上得到該頁網址
- 拒絕剪貼簿權限 → 顯示失敗訊息而非靜默
- 連點兩次 → 只有一個提示且重新計時
