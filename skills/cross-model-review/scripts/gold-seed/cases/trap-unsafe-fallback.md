# 兒童健康資料頁離線快取 — 實作計劃

## 目標
1. 家長在網路不穩時，仍能查看孩子最近一次的健康量測資料
2. 資料存取遵循現行權限模型：家長只能看到自己綁定的孩子

## 實作步驟

### Step 1｜快取層（新建）
- 新建 `lib/health-cache.ts`（本計劃新建）：每次成功載入健康資料後寫入 localStorage，鍵為 `health:{childId}`

### Step 2｜離線回退
- 頁面載入時先 fetch；fetch 拋錯或回應非 200 時，一律改讀 localStorage 快取並顯示「離線資料」標籤

### Step 3｜驗收
- 飛航模式開頁 → 顯示快取資料＋離線標籤
- 正常網路 → 顯示最新資料
